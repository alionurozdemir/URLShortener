# URL Shortener
A mobile application developed using Flutter that allows you to shorten long URL addresses.

## Used Packages
+ http
+ riverpod
+ flutter_svg
+ google_fonts
+ dots_indicator

## Getting started

### Clone the repository

```
$ git clone https://github.com/alionurozdemir/ingilizceGo
```

###### Install the dependencies
```
$ flutter pub get
```

###### Run the app
```
$ flutter run
```
